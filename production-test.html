📋 Test Talimatları

  1. Extension'ı yükleyin - Chrome'da chrome://extensions/ adresine gidin
  2. Bu sayfadaki herhangi bir elemento sağ tıklayın
  3. "Erişilebilirlik Analizi" seçeneğini tıklayın
  4. AI analiz sonuçlarını bekleyin

Not: Eğer context menu'da seçenek görünmüyorsa, extension'ı yeniden yükleyin ve sayfayı yenileyin.

✅ Erişilebilir Elementler

Bu elementler WCAG standartlarına uygun:

İyi örnek: Açıklayıcı aria-label ile buton

İyi örnek: Alt text ile resim

İyi örnek: Label ile bağlanmış input

🚨 Problemli Elementler

Bu elementlerde erişilebilirlik sorunları var:

Problem: Alt text eksik resim - ekran okuyucular bunu okuyamaz

Problem: WCAG kontrast oranı yetersiz (4.5:1 olmalı)

Tıklanabilir div (klavye erişimi yok)

Problem: Klavye ile erişilemeyen etkileşimli element

Problem: Label eksik input - ne için olduğu belirsiz

Zayıf kontrast link

Problem: Link rengi yeterince belirgin değil
